Images

Images can be responsive to fit the parent's width,and also can be customized ti be round shape.

Example of Responsive Image

You can add class img-responsive to make your image fit the width of container.Its height will get adjusted by keeping the aspect ratio,you will have to crop the image.Add class img_responsive

Example of Round Image

You can add class img_roundto make your image round shaped.

Example of Sqaure Image

You can add class img_squareto make your image round shaped.
